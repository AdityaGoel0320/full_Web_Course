Cheat Sheet - Markdown


Markdown Cheat Sheet

Basic Syntax

  • To create a heading

    # H1
    ## H2
    ### H3

  • To bold a text

    **bold text**

  • To italicize a text

    *italicize text*

  • To create a blockquote

    > blockquote

  • To create an ordered list

    1. First item
    2. Second item
    3. Third item

  • To create an unordered list

    - First item
    - Second item
    - Third item

  • To create a code snippet

    `code`

  • To create a horizontal rule

    ---

  • To create a link

    [title](https://www.example.com)

  • To add an image

    ![alt text](image.jpg)

Extended Syntax

  • To create a table

    | Syntax | Description |
    | ----------- | ----------- |
    | Header | Title |
    | Paragraph | Text |

  • To create a fenced code block

    ```
    {
    "firstName": "John",
    "lastName": "Smith",
    "age": 25
    }
    ```

  • To create a footnote

    Here's a sentence with a footnote. [^1]
    [^1]: This is the footnote.

  • To add a custom heading ID

    ### My Great Heading {#custom-id}

  • To create a definition list

    term
    : definition

  • To strikethrough a text

    ~~The world is flat.~~

  • To create a task list

    - [x] Register for hacktoberfest
    - [ ] Learn Git and GitHub
    - [ ] Contribute to open-source

  • To add an emoji

    :smile:

  • To highlight a text

    ==highlighted text==

  • To create a subscript

    H~2~O

  • To create a superscript

    9^th^
